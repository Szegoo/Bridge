## Pokretanje servera:
  - Sta je potrebno?
    - nodejs
    - npm
  1. ```shell
     git clone https://github.com/Szegoo/Bridge.git
     ```
  2. ```shell
     cd Bridge
     ```
  4. ```shell
     npm i
     ```
  3. ```shell
     npm start
     ```
